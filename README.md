# Hướng dẫn sử dụng
1. Ở mục releases của repo này, tải "Tool DKHP UIT 1.2" về
2. Chạy file installer vừa tải
3. Cài xong, sẽ thấy shortcut "Tool DKHP UIT" ở desktop
4. Mở lên và dùng thôi

# Các tính năng chính
- Tự động đăng ký các học phần đã cài sẵn
- F5 liên tục cho đến khi đăng ký được học phần
